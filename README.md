# 🧠 Machine Learning Project - EDA, Regresión y Clasificación

## 📌 Descripción
Proyecto de análisis exploratorio de datos (EDA) y aplicación de modelos de Machine Learning utilizando datasets reales de OpenML.

Se trabajó con:
- 🏥 BUPA Liver Disorders (ID:8)
- 💳 German Credit Dataset (ID:31)

## 🔍 Actividades realizadas
- Limpieza y preprocesamiento de datos
- Eliminación de outliers (IQR)
- Análisis cuantitativo y cualitativo
- Visualización (Boxplots, Heatmaps, Scatter)
- Modelos de Machine Learning:
  - Logistic Regression
  - Random Forest
  - Linear Regression

## 📊 Métricas utilizadas
- Accuracy
- Precision, Recall, F1-score
- R²
- MSE
- Matriz de confusión

## 🛠️ Tecnologías
Python | Pandas | NumPy | Seaborn | Matplotlib | Scikit-learn
